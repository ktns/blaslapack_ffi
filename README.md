# BlasLapackFFI

FFI for BLAS/LAPACK routines

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'blaslapack_ffi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install blaslapack_ffi
